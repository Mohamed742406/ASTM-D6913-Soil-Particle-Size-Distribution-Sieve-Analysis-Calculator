## 🖥️ طريقة الاستخدام

### 1. بيانات المشروع
أدخل بيانات المشروع: اسم العميل، اسم المشروع، الموقع، رقم العينة، تاريخ الاختبار، الفني، وظروف البيئة.

### 2. اختر نوع النخل
اختر أحد التبويبات الثلاثة:
- **فصلين** — نخل مركب بنقطتَي فصل
- **فصل واحد** — نخل مركب بنقطة فصل واحدة
- **نخل بسيط** — تحليل النخل القياسي

### 3. أدخل بيانات الاختبار
- حدد أحجام مناخل الفصل (Sep.1 و Sep.2)
- أدخل الأوزان الرطبة والجافة لكل جزء
- أدخل الأوزان المحتجزة على كل منخل

### 4. احسب
اضغط **احسب** للحصول على:
- النسبة المئوية المحتجزة التراكمية والمارة لجميع المناخل
- 1st CSCF و 2nd CSCF
- D10 و D30 و D60 و Cu و Cc
- تصنيف التربة USCS و AASHTO

### 5. تصدير النتائج
- **طباعة التقرير** — ينشئ تقريراً بحجم A4 جاهزاً للطباعة
- **حفظ الجلسة** — يُصدّر جميع البيانات بصيغة `.json`
- **تحميل الجلسة** — يُعيد تحميل جلسة محفوظة مسبقاً

## 📐 المعادلات الأساسية

### معامل تصحيح العينة المركبة (CSCF)

**1st CSCF:**
$$\text{1stCSCF} = \frac{S_{M_d}}{FP_{M_d}}$$

**2nd CSCF:**
$$\text{2ndCSCF} = \text{1stCSCF} \times (1 - \text{FCMR}_{\text{Sep.2}})$$

### النسبة المئوية المحتجزة التراكمية المجمعة (CMR)

$$\text{CMR}_i = \frac{\text{محتجز CP التراكمي} + \text{أجزاء SubS}}{S_{M_d}} \times 100$$

### النسبة المئوية المارة

$$\%P = \left(1 - \frac{\text{CMR}}{S_{M_d}}\right) \times 100$$

### معاملات التدرج

$$C_u = \frac{D_{60}}{D_{10}}, \quad C_c = \frac{D_{30}^2}{D_{60} \times D_{10}}$$
